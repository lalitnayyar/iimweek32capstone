## Project Overview

ChurnGuard AI is a comprehensive capstone project that develops a strategic business plan for an artificial intelligence-powered customer churn prediction platform. This project demonstrates the application of AI/ML technologies to solve real-world business challenges while addressing market opportunities, financial viability, and implementation considerations.

The project deliverables include a professional presentation, comprehensive business plan report, Python analysis code, interactive GitHub Pages website, and complete documentation.

## Getting Started

### Prerequisites

- Web browser (Chrome, Firefox, Safari, Edge)
- Python 3.7+ (for running analysis code)
- Git (for cloning repository)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/lalitnayyar/iimweek32capstone.git
   cd iimweek32capstone
   ```

2. **Open the website locally:**
   ```bash
   # Using Python's built-in server
   python3 -m http.server 8000
   
   # Then visit: http://localhost:8000
   ```

3. **Run the analysis code:**
   ```bash
   python3 churnguard_analysis.py
   ```

### Viewing the Presentation

1. **Online:** Visit `https://lalitnayyar.github.io/iimweek32capstone`
2. **Locally:** Open `churnguard_presentation/slide_01_title.html` in a web browser
3. **Navigation:** Use arrow keys or click navigation buttons to move between slides
