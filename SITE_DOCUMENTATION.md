## Presentation Viewer

### Accessing the Presentation

1. Click "Presentation" in main navigation
2. Slide 1 (Title Slide) loads automatically
3. Use navigation controls to browse slides

### Navigation Controls

**Keyboard Navigation:**
- **Right Arrow:** Next slide
- **Left Arrow:** Previous slide
- **Home:** First slide
- **End:** Last slide
- **Number + Enter:** Jump to specific slide

**Mouse Navigation:**
- **Next Button:** Click arrow to go to next slide
- **Previous Button:** Click arrow to go to previous slide
- **Slide Counter:** Click to see all slides
- **Thumbnail View:** Click to jump to specific slide

### Slide Information

Each slide displays:
- **Title:** Main heading of the slide
- **Content:** Detailed information and visuals
- **Images/Charts:** Data visualizations
- **Footer:** Student name, email, course info, slide number

### Slide Features

**Zoom Controls:**
- Use browser zoom (Ctrl/Cmd + or -)
- Zoom to 200% for detailed viewing
- Zoom to 50% for overview

**Full Screen Mode:**
- Press F11 for full screen
- Ideal for presentations
- Press F11 again to exit

**Print Slide:**
- Use browser print (Ctrl/Cmd + P)
- Select "Save as PDF" for digital copy
- Print individual slides or entire presentation

### Slide Navigation Tips

- **Quick Navigation:** Use keyboard arrows for fast browsing
- **Specific Slide:** Jump to specific slide using number keys
- **Overview:** Use thumbnail view to see all slides at once
- **Presentation Mode:** Use full screen for immersive viewing

## Interactive Elements

### 1. Bookmarks Feature

**Purpose:** Save important slides for quick reference

**How to Use:**
1. Navigate to slide you want to bookmark
2. Click "Bookmark" button (star icon)
3. Slide is added to bookmarks
4. Access bookmarks from sidebar menu
5. Click bookmarked slide to jump to it

**Managing Bookmarks:**
- View all bookmarks in "Bookmarks" section
- Remove bookmark by clicking star again
- Bookmarks persist across sessions
- Export bookmarks as list

### 2. Notes Feature

**Purpose:** Add personal notes to slides

**How to Use:**
1. Navigate to slide
2. Click "Notes" button
3. Type your notes in text box
4. Click "Save" to save notes
5. Notes appear below slide

**Managing Notes:**
- Edit notes by clicking "Edit" button
- Delete notes by clicking "Delete" button
- View all notes in "My Notes" section
- Export notes as document

### 3. Copy Protection

**Purpose:** Protect content from unauthorized copying

**Features:**
- Text cannot be selected and copied
- Right-click copy is disabled
- Keyboard shortcuts (Ctrl+C) don't work
- Content remains readable and accessible

**Accessing Content:**
- Read content directly on website
- Take screenshots for personal use
- Print slides or documents
- Download PDF versions where available

### 4. Search Functionality

**Purpose:** Find specific information quickly

**How to Use:**
1. Click search icon in navigation
2. Enter search term
3. Press Enter or click Search button
4. Results appear below search box
5. Click result to navigate to content

**Search Tips:**
- Use specific keywords for better results
- Search across all content types
- Filter results by content type
- Save frequent searches

## Troubleshooting

### Issue: Website Not Loading

**Problem:** Website doesn't load or shows error

**Solutions:**
1. Clear browser cache (Ctrl+Shift+Delete)
2. Hard refresh page (Ctrl+Shift+R)
3. Try different browser
4. Check internet connection
5. Disable browser extensions

### Issue: Slides Not Displaying

**Problem:** Slides appear blank or don't load

**Solutions:**
1. Refresh page (F5 or Ctrl+R)
2. Clear browser cache
3. Try different browser
4. Check JavaScript is enabled
5. Check internet connection

### Issue: Images Not Loading

**Problem:** Images show as broken links

**Solutions:**
1. Refresh page
2. Clear browser cache
3. Check internet connection
4. Disable ad blocker
5. Try different browser

### Issue: Slow Performance

**Problem:** Website loads slowly or is sluggish

**Solutions:**
1. Close other browser tabs
2. Disable browser extensions
3. Clear browser cache
4. Check internet connection
5. Try different browser
6. Restart browser

### Issue: Navigation Not Working

**Problem:** Menu items don't work or links are broken

**Solutions:**
1. Refresh page
2. Clear browser cache
3. Check JavaScript is enabled
4. Try different browser
5. Check internet connection

### Issue: Mobile Menu Not Opening

**Problem:** Hamburger menu doesn't open on mobile

**Solutions:**
1. Refresh page
2. Rotate device to landscape and back
3. Clear browser cache
4. Try different mobile browser
5. Disable browser zoom

### Issue: Print Not Working

**Problem:** Can't print slides or documents

**Solutions:**
1. Check printer is connected
2. Try print to PDF instead
3. Check browser print settings
4. Disable browser extensions
5. Try different browser
